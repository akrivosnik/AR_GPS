Detailed Description

The ARUI MonoBehaviour manages user interface interactions specific to the AR experience. In particular, it handles the functionality of the Map button, allowing users to exit the AR scene and return to the main map or overview. This script demonstrates how to connect UI elements to application logic, enabling seamless transitions between AR and non-AR views in tourism or navigation apps.

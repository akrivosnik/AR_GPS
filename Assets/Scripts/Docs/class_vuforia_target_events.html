VuforiaTargetEvents Class Reference

This MonoBehaviour acts as an event dispatcher for Vuforia target status changes. It listens for changes in the tracking status of Vuforia ObserverBehaviours (such as Image Targets or Model Targets) and exposes these changes as C# events that other scripts can subscribe to. This enables decoupled and modular handling of AR target detection, loss, and updates, which is essential for building interactive AR experiences where UI or logic must respond to target visibility. Detailed Description

This MonoBehaviour acts as an event dispatcher for Vuforia target status changes. It listens for changes in the tracking status of Vuforia ObserverBehaviours (such as Image Targets or Model Targets) and exposes these changes as C# events that other scripts can subscribe to. This enables decoupled and modular handling of AR target detection, loss, and updates, which is essential for building interactive AR experiences where UI or logic must respond to target visibility.

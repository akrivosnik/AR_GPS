Detailed Description

A Unity ScriptableObject that serves as a persistent data container for a collection of places of interest. This class is designed to be used as a centralized database for storing and managing multiple "PlaceOfinterest" entries, which represent real-world locations with associated metadata such as name, description, address, coordinates, and an icon. The ScriptableObject can be edited in the Unity Editor and referenced by other scripts at runtime, enabling efficient data-driven workflows for location-based AR or tourism applications.

PlaceOfinterest Class Reference

A serializable data class that encapsulates all relevant information about a single place of interest. This class is intended to represent real-world locations in AR tourism or mapping applications, providing both geospatial coordinates and descriptive metadata for each location. Detailed Description

A serializable data class that encapsulates all relevant information about a single place of interest. This class is intended to represent real-world locations in AR tourism or mapping applications, providing both geospatial coordinates and descriptive metadata for each location. Each instance can be stored in a "PlacesOfinterest" ScriptableObject and referenced at runtime.
